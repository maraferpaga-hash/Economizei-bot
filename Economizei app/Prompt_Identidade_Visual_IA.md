## Prompt 1 — Logo Principal

> **Melhor ferramenta:** Ideogram 2.0 (melhor texto em logo) ou Adobe Firefly

```
Design a minimalist logo for a Brazilian fintech app called "Economizei" 
(Portuguese for "I saved money"). 

Concept: A clean checkmark (✓) integrated with a subtle downward arrow, 
symbolizing savings going down (spending) and confirmation. 
OR: The letter "E" stylized with a downward-pointing arrow embedded in it.

Style:
- Flat design, no gradients, no shadows
- Two-color maximum: deep forest green (#1A6B3C) and near-black (#0F1F14)
- Geometric, modern, confidence-inspiring
- Works at 32x32px (app icon) and large formats

Do NOT include:
- Shopping carts (too generic)
- Dollar signs or coins
- Piggy banks
- Complex illustrations

Output: Show 3 logo variations — icon only, icon + wordmark horizontal, 
icon + wordmark stacked. White background. Clean presentation.
```

## Prompt 2 — Paleta de Cores Aplicada (Brand Sheet)

> **Melhor ferramenta:** DALL-E 3 ou Midjourney v6

```
Create a brand color palette presentation sheet for a Brazilian personal finance 
WhatsApp bot called "Economizei". 

Color system:
- Primary: Deep forest green #1A6B3C (trust, stability, money)
- Accent: Lime green #72C442 (action, celebration, "you saved!")  
- Text: Near-black #0F1F14 (warm, not harsh)
- Background: Light sage #F4F7F4 (neutral with green undertone)
- Alert: Warm red #C94040 (spending above average — used sparingly)

Show the colors as:
1. Rounded square swatches with hex codes labeled
2. Sample UI card mockup using the palette (a simple "expense summary" card)
3. The palette on both light and dark backgrounds

Style: Clean, modern, professional. Inspired by Nubank and Inter Bank aesthetics 
but warmer. Sans-serif typography throughout.
```

## Prompt 3 — Template de Post TikTok/Reels (Frame estático)

> **Melhor ferramenta:** Canva (com os parâmetros abaixo) ou DALL-E 3

```
Design a vertical social media post template (1080x1920px, 9:16 ratio) 
for a Brazilian fintech brand called "Economizei".

Layout:
- Top 20%: brand logo top-left, small and clean
- Center 60%: large bold provocative question text in Portuguese, 
  example: "Você sabe quanto gastou no mercado esse mês?"
  Typography: Inter Black or similar geometric sans-serif, 
  white text on dark green background OR dark text on light background
- Bottom 20%: subtle CTA area with WhatsApp icon and "Link na bio"

Color: Primary deep green #1A6B3C background with white (#FFFFFF) text
Accent elements: thin lime green (#72C442) line or geometric shape

Style:
- Minimal — text is the hero, not decoration
- No stock photos, no emojis in the design itself
- Feels like a confident, smart Brazilian brand (think: Nubank meets 
  a neighborhood friend who knows things)

Show 2 variations: dark background version and light background version.
```

## Prompt 4 — Mockup de WhatsApp (Prova Social Visual)

> **Melhor ferramenta:** DALL-E 3 ou criar manualmente no Canva com este briefing

```
Create a realistic smartphone mockup showing a WhatsApp conversation 
for a Brazilian personal finance bot called "Economizei".

The conversation shown (in Portuguese):
- User sends: [photo of a supermarket receipt]
- Bot replies with a clean formatted message:
  "✅ Compra registrada!
   🏪 Supermercado Extra — 17/05/2026
   💰 Total: R$ 187,40
   
   Seus gastos este mês: R$ 412,80
   📊 Você está 8% abaixo da sua média. Bom trabalho! 👊"

Phone style: Modern Android/iPhone, clean, neutral color
Background behind phone: Soft sage green (#F4F7F4) or white
Lighting: Soft, product-photography style

The message design should feel real — use WhatsApp's actual visual style 
(green bubbles for received, no user bubble shown). 
Add a subtle brand element (Economizei logo small, bottom of image).
```

## Prompt 5 — Avatar / Profile Picture das Redes Sociais

> **Melhor ferramenta:** Ideogram ou Midjourney

```
Design a social media profile picture for "Economizei", a Brazilian 
WhatsApp-based personal finance bot.

Requirements:
- Square format, 800x800px minimum
- Works as a circle (Instagram/TikTok crop)
- Simple enough to be recognizable at 50x50px
- Icon-forward: the logo mark only, no wordmark

Concept options to explore:
Option A: Bold checkmark (✓) in lime green (#72C442) on deep forest green (#1A6B3C) circle background
Option B: Stylized "E" letter with downward arrow in white on deep green circle
Option C: Simple receipt/ticket icon reduced to geometric minimal lines, white on green

Style: Flat, bold, no gradients. High contrast. Instantly readable. 
Do NOT use: complex illustrations, photographs, gradients, multiple colors beyond 2.

Show all 3 options side by side for comparison.
```

## Prompt 6 — Thumbnail de TikTok (Cover Frame)

> **Melhor ferramenta:** DALL-E 3 ou Canva

```
Design a TikTok video cover thumbnail template for "Economizei" brand.

Format: 1080x1920px vertical
Style: Bold, high-contrast, scroll-stopping

Layout:
- Large provocative number or question dominates 70% of frame
  Example texts: "R$ 847" / "Você sabia?" / "3x mais caro"
- Brand logo small, top-right corner
- Brand color: deep green (#1A6B3C) background
- Text: White, Inter Black font, very large

The feeling: a friend texting you something important. 
Urgent but not alarming. Smart, not salesy.

Show 3 thumbnail variations with different text examples.
```

### Dica de workflow
1. Gerar o **logo primeiro** (Prompt 1) — ele define tudo
2. Aprovar 1 variação do logo
3. Usar a paleta confirmada nos demais prompts
4. Criar o **avatar** (Prompt 5) e configurar os perfis do TikTok e Instagram
5. Criar o **template de post** (Prompt 3) no Canva usando os arquivos do logo
6. Ter o **mockup de WhatsApp** (Prompt 4) pronto para o primeiro post de demo

### Variáveis para personalizar
Antes de usar os prompts, você pode substituir os valores entre `[ ]`:
- `[photo of a supermarket receipt]` → usar foto real do cupom
- Os exemplos de mensagens do bot → usar saída real do seu bot
- Os números de exemplo → substituir por dados reais quando tiver usuários
